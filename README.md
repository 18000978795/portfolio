# 个人项目作品集

这里整理了我用于求职展示的全栈 / 前端项目。项目覆盖前端页面开发、后端接口设计、数据库持久化、工程化配置和 README 文档交付。

## 项目列表

### 1. 校园学习打卡与任务管理系统

仓库地址：[campus-task-manager](https://github.com/18000978795/campus-task-manager)

一个面向学生学习计划管理场景的全栈项目，包含任务 CRUD、学习打卡、统计看板、Express API、MySQL 持久化和完整 README 交付说明。

技术栈：

- Vue 3
- TypeScript
- Vite
- Node.js
- Express
- MySQL

项目亮点：

- 前后端分离结构
- 任务管理与学习打卡完整 CRUD
- MySQL 数据持久化
- 统计看板与筛选交互
- 适合 GitHub 求职作品展示

### 2. AI Video Studio

仓库地址：[ai-video-studio](https://github.com/18000978795/ai-video-studio)

一个 AI 视频生成 SaaS / 桌面应用 MVP，支持图片上传、浏览器内图片编辑、视频生成任务、任务轮询和历史素材库。当前版本使用 Mock Provider 模拟视频生成流程，保留了接入真实模型 API 的扩展结构。

技术栈：

- Next.js
- React
- TypeScript
- Electron
- Prisma
- Fabric.js

项目亮点：

- 图片上传与编辑流程
- 视频生成任务模型
- 本地桌面应用模式
- 历史素材库
- 适合作为前端工程化和产品原型能力展示
